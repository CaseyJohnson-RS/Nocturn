# Сервис Авторизации

## Описание

Это сервис авторизации, который (для остальных сервисов) только выдает JWT токен. Однако на нём куча функций, например, подтверждение почты, слежка за компрометацией и подобное.

Запуск непосредственно на хосте

1. Если у вас не установлен пакетный менеджер uv, то устанавливаем:
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```
2. Восстанавливаем окружение (запускаем команду в корне проекта)
```bash
uv sync
```
3. Активируем окружение
```bash
source .venv/bin/activate  # Linux/macOS
# или
.venv\Scripts\activate  # Windows
```
4. Запускаем ASGI
```bash
uvicorn app.main:app --host 127.0.0.1 --port 8000 --reload --proxy-headers
```
